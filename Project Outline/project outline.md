# 多源电力市场

## 模型：负荷预测模型

负荷预测就是对电力部门日常编排**调度计划、供电计划、交易计划，制定相符的检修和运行计划**等，首先需要对电网的重要影响因素进行综合分析和研究， 如电网的系统**运行特性、经济情况、气候条件**等。然后对历史负荷**数据的变化规律**进行分析并找出**各影响因数与负荷之间**的关系，最后在实际的基础上依据数学 理论和计算机编程等建立预测模型，实现准确的负荷预测
